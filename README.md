# VESS
Visual Evaluation of Soil Structure (VESS)

<img src="/docs/VESS-Icon2.png" width="200">

## Installation
```
git clone https://github.com/MichaelPolla/VESS.git
cd VESS
npm install
```
Run :  
`ionic run android`  
or    
`ionic serve` (not all functionalities)
